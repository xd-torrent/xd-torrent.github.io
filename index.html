XD: Anonymous filesharing client

Don't pay for a VPN again, use XD instead!

Features:

  • P2P file sharing with BitTorrent protocol
  • All connections are encrypted and anonymized with I2P (Invisible Internet Protocol)
  • Web and console user interfaces
  • Simple JSON-RPC API for making custom interfaces and scripts
